# 前端开发

前端开发是创建网站和 Web 应用程序的用户界面和用户体验的过程。主要使用 HTML、CSS 和 JavaScript 技术。

## 学习路线

```
HTML → CSS → JavaScript → Vue/React → 工程化
```

## 核心技能

### 1. HTML
- 语义化标签
- 表单和输入
- 多媒体元素
- SEO 基础

### 2. CSS
- 选择器和优先级
- 盒模型和布局
- Flexbox 和 Grid
- 响应式设计
- 动画和过渡

### 3. JavaScript
- 基础语法
- DOM 操作
- 事件处理
- 异步编程
- ES6+ 新特性
