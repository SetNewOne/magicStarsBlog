# 后端开发

后端开发主要负责服务器端的逻辑处理和数据存储。

## 学习路线

```
Java/Python → 数据库 → 设计模式 → 框架 → 微服务
```

## 核心技能

### 1. Java
- 基础语法和面向对象
- 集合框架
- IO/NIO
- 多线程
- JVM 原理
- Spring/Spring Boot
- MyBatis/Hibernate

### 2. Python
- 基础语法
- 数据类型和结构
- 函数和模块
- Flask/Django/FastAPI
- 数据分析（Pandas/NumPy）

## 常见应用场景

- 企业级应用开发
- 微服务架构
- 大数据处理
- 人工智能/机器学习
- 自动化脚本
