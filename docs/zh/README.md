## 🚀 快速开始

```bash
# 克隆博客源码
git clone https://github.com/SetNewOne/magicStarsBlog.git

# 进入目录
cd magicStarsBlog

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 [http://localhost:8080](http://localhost:8080) 预览博客
