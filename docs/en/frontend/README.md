# Frontend Development

Frontend development creates the user interface and user experience of websites and web applications using HTML, CSS, and JavaScript.

## Learning Path

```
HTML → CSS → JavaScript → Vue/React → Build Tools
```

## Core Skills

### 1. HTML
- Semantic tags
- Forms and inputs
- Multimedia elements
- SEO basics

### 2. CSS
- Selectors and specificity
- Box model and layout
- Flexbox and Grid
- Responsive design
- Animations and transitions

### 3. JavaScript
- Basic syntax
- DOM manipulation
- Event handling
- Asynchronous programming
- ES6+ features
